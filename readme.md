### Instalação

Se não possui Yarn
```
npm install --global yarn
```

para instalar as dependencias
```
yarn
```

para iniciar o app
```
yarn start:dev
```

se quer um fresh start, delete o dev.sqlite e use o comando
```
yarn knex migrate:latest
```
